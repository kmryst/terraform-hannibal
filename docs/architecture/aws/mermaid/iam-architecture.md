# IAM権限構成

IAM Role 一覧の正本は [iam-management.md](../../../operations/iam-management.md) です。

この文書では IAM Role / Policy の Mermaid 図を管理しません。Role、Assume元、権限方針、管理場所は表形式の Role カタログを正本にします。図は更新漏れが起きやすく、GitHub Actions OIDC や PR plan 用 Role の追加時に実態とずれやすいためです。

PR terraform plan 用 Role / OIDC 権限の詳細設計補足は [pr-terraform-plan-role-design.md](../../../operations/pr-terraform-plan-role-design.md) を参照します。
